## Installation
with docker

1. Clone the repo
   ```sh
   git clone https://github.com/lakshitha212/to-do.git
   ```
2. ``` cd to-do ```
3. ``` docker image prune -f  ```
4. ``` docker-compose up --force-recreate --build -d  ```
